3D Scanning and printing

I. 3D scanning

scanning using picza-modella

The aim of this project for 3D scanning assignment is to study the complex form of nature and capture it with 3D printer. I wanted to make something which can not be easily modeled in 3D softwares.  So for this I found the natural objects very interesting.
The machine doesn’t read the bottom part of any scanning object, so i stick it with silicon in-order not to be shaken while on the scanning process.

For this reason it would be better to choose top and side views which I chose is more like conical. I actually scanned two shells, but this one is more convenient for the project.
The things I needed to consider while sending to the machine model la is the maximum and minumum height,maximum and maximum and width, the tool diameter, the x,y,z position on the modella board. After setting all this on picza I started scanning., Since the shell I used to scan was so small the scanning process was not so long, which is 2 hours and 15 minutes.

After finishing the 3D scanning on picza, I export it as STL file. Then on 3D max studio I opened the STL file I saved but not import. I tried to fill the scanned form by editable poly conversion and extruding the below part, I fuse it and try to see the different views and make various
exercises and choose 3/4 for the final one.
**

top part scanned

**
I don’t want to use mesh smooth effect because it will lack the irregular shape found on the object. I focus on the forms and textures, because besides stl files doesn’t read colors, the project is all about-3D scanning and printings. I tried to create different arrangements by joining and cutting inorder to print it with 3D.

Options

From 3d scanning

From 3d scanning

*When carefully studied this shell has tremendous shape which cannot stated, but after studying it, we will get various designs
which can be transformed and converted to pattern or formal design. The scanning process has been taken already in digitalis, this makes it easy for me to discover
the detail pattern scales for each bites.

From 3d scanning

By exploring the different forms I tried how much is possible to scan and get the textures. The bits on the pattern of the top of the shell has an amazing structure which I divided and started studying it.

After finishing the modeling, the next step is to export the file as Stl file. The model I choose to print is the solid one. But for economizing the powder of the 3D printer,  it would be preferable to make it hollow.

After sending the 3D print to the 3D printer for 24 hours/i.e. with other colleagues models/,  I carefully brushed the model like archeological discovery and vacuum it with the vacume in order to clean every dusty particle on it.  This is because 3D printer here in Fab BcN/z-corp works upfusing layer powder over powder.

At last I sprayed water on it to make it strong and fixed. Finally I have the prototype in hand (digitally fabricated).
